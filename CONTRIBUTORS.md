## How to Become a Contributor

We welcome contributions from everyone! Here's how you can contribute:

1. **Code Contributions**: Submit pull requests for bug fixes, features, or improvements
2. **Documentation**: Help improve our documentation
3. **Bug Reports**: Report bugs and issues you encounter
4. **Feature Requests**: Suggest new features and enhancements
5. **Community Support**: Help other users in discussions and issues
6. **Testing**: Test new features and report feedback
7. **Translation**: Help translate the application (coming soon)

See our [Contributing Guide](CONTRIBUTING.md) for detailed information.

## Recognition Criteria

Contributors are recognized based on their contributions:

- **Core Contributors**: Regular contributors with significant impact on the project
- **Contributors**: Anyone who has had a pull request merged or provided significant help
